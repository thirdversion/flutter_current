CurrentValidationIssue

CurrentValidationIssue class final

Describes a validation failure without tying it to a specific locale.

Validation rules should return this object instead of already-localized display text. Widgets can later resolve the issue into a localized string using the active CurrentValidationIssueTextResolver.
