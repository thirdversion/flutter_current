CurrentDateTimeProperty

CurrentDateTimeProperty class

An CurrentProperty with similar characteristics as a an ordinary dart DateTime object

Then underlying DateTime value cannot be null

When the value of this changes, it will send a CurrentStateChanged event by default. This includes automatically triggering a UI rebuild.
