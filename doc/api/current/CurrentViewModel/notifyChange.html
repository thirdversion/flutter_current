Inform the bound CurrentState that the state of the UI needs to be updated with a single event.

This is a convenience method that allows you to avoid having to create a list when you only have one event to notify. NOTE: Although you CAN call this method manually, it's usually not required. Updating the value of an CurrentProperty will automatically notify the UI to update itself. But if you have a custom event that you'd like to notify the UI of, you can use this method to do so.

Implementation

void notifyChange(CurrentStateChanged event) {
  notifyChanges([event]);
}
