CurrentNullableStringProperty

CurrentNullableStringProperty class

An CurrentProperty with similar characteristics as a dart String object

The underlying value can be null.

You can easily check for null by accessing the isNull or isNotNull properties.

When the value of this changes, it will send a CurrentStateChanged event by default. This includes automatically triggering a UI rebuild.
