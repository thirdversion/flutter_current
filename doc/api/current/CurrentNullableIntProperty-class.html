CurrentNullableIntProperty

CurrentNullableIntProperty class

An CurrentProperty with similar characteristics of dart int objects

The underlying value can be null.

If the underlying value of this is null and an arithmetic operator is called on this, it will throw a CurrentPropertyNullValueException.

You can easily check for null by accessing the isNull or isNotNull properties.

When the value of this changes, it will send a CurrentStateChanged event by default. This includes automatically triggering a UI rebuild.

Example

final age = CurrentNullableIntProperty();

if (age.isNull)
{
  print("I'm Null");
}

Other Usages Examples


final age = CurrentNullableIntProperty();

print('${age.add(5)}'); //throws CurrentNullValueException because no value has been set yet.

age(10)

print('${age.subtract(5)}'); //prints 5
