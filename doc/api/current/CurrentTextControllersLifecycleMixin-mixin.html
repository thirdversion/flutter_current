CurrentTextControllersLifecycleMixin

CurrentTextControllersLifecycleMixin<TWidget extends CurrentWidget<CurrentViewModel>, TViewModel extends CurrentViewModel> mixin

A mixin for managing the lifecycle of CurrentTextControllers in a CurrentState.

This mixin initializes controllers on the first dependency pass and re-runs bindCurrentControllers when the widget updates so controllers can rebind to new properties when needed.

Must implement bindCurrentControllers. This is where you should initialize your CurrentTextControllers.
