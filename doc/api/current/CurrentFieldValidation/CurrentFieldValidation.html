Creates a validator for the provided property.

The optional rules are evaluated in the order supplied.

Validators created through CurrentPropertyValidationExtensions.createValidation are registered against the property automatically.

If the property already belongs to a CurrentViewModel, the validator attaches immediately. Otherwise it attaches the next time the property is assigned to a view model.
